# A/B TEST

## When can you use A/B testing：
- 对照组和实验组（是否容易设置两组）
- 明确的归因（归因是否清晰？）
- 排除其他影响（设备？人群？）
- 时间成本（回馈是否过长？）

## Other methods you can use：

- Customer funnel

- click-through-rate and click-through-probability

- if N * p_hat > 5, then you can assume it is normal.
you should also check N * (1 - p_hat) > 5 satisfied.
